<Question>

Create and test an XHTML document that describes nested ordered list of cars. The outer list must have 3 entries: compact, midsize and sports. Inside each of these 3 lists there must be 2 sublists of body styles. The compact and midsize car sublists are two door and four door; the sports car sublist are coupe and convertible. Each body-style sublist must have atleast 3 entries, each of which is the make and model of a particular car that fits the category. The outer list must use uppercase Roman numerals, the middle list must use the upper case letters, and the inner list must use Arabic numerals. The background color for the compact car list must be pink; for midsize list it must be blue; for sports car list, it must be red. All of the styles must be in a document-style sheet.
